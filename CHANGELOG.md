# 修改歷史 Change Log

### V0.2.1 (2025.07.23)
同源詞映射規則重寫：改用橋字驗證法替代統計對應表。

- 橋字驗證：用字表中已知字證明聲母、韻母、聲調三維度的規律性對應
- 排除 □（有音無字）參與橋驗證
- 最低橋字數 3，過濾偶發對應
- 貪心團聚類替代錨點法，確保組內每對都通過驗證
- 義類合併：躲藏與捉迷藏統一為 HIDE
- 手動修正：HIDE_tsʰia1 組合併 ia/ia_2，保留 tone 1/5，補入石門、沅陵七甲坪

### V0.2.0 (2025.07.23)
新增同源詞識別功能（實驗性）。

基於漢字音典字表數據，構建語音對應表，自動識別跨方言同源詞：
- IPA 解析器：將讀音拆分為聲母、韻母、聲調
- 語音對應表：統計方言間韻母對應規律
- 同源詞聚類：基於強對應關係聚類，採用錨點法避免過度合併
- 支持手動覆蓋（cognate_manual 表，重建時保留）

搜索結果中，若某方言點讀音屬於同源詞組，可點擊「同源 N 詞」展開查看同組其他方言的讀音。

目前覆蓋「躲藏」「捉迷藏」兩個義類，後續將擴展更多義類。

### V0.1.4 (2025.07.23)
卡片內按方言點分組，同一方言點的多個讀音合併展示。

方言名獨佔一行（帶圓點），下方每條讀音獨立成行：IPA 居左、註釋居右。

### V0.1.3 (2025.07.23)
修復升級後資料庫未更新導致查詢崩潰的問題。

新增資料庫版本檢查機制：檢測到舊版資料庫時自動刪除並從 assets 重新拷貝，確保 schema 升級（排序列）生效。

### V0.1.2 (2025.07.23)
搜尋結果按漢字音典「音典排序」排列各方言點讀音。

資料庫新增「排序」列，存儲各方言點的音典排序號（如 I9B-008），卡片內各方言點按此序號順序展示。

### V0.1.1 (2025.07.23)
搜尋結果按字組分組：相同字組在不同方言點的讀音合併到同一張卡片展示。

卡片重排為兩段式結構：頂部顯示字組標題及方言點數，下方逐行列出各方言點讀音。

### V0.1.0 (2025.07.22)
湘典首個版本。

基於漢字音典（MCPDict）字表資料，構建湖南及周邊地區方言讀音資料庫。

功能：
- 搜字：輸入漢字查詢各方言點讀音
- 搜音：輸入 IPA 讀音反查漢字
- 搜釋義：通過釋義關鍵詞反查漢字
- 搜尋結果以卡片展示，含漢字、方言名稱、IPA 讀音及註釋

資料庫：
- 收錄 464 個方言點
- 117,645 組同音字
- 逾 60 萬條讀音記錄
- 覆蓋湘語全部五片及周邊地區

技術：
- Kotlin + Jetpack Compose + Material 3
- SQLite FTS5 全文檢索，離線使用
- 最低支持 Android 7.0（API 24）
