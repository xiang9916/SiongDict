# 修改歷史 Change Log

### V0.1.4 (2025.07.23)
卡片內按方言點分組，同一方言點的多個讀音合併展示。

方言名獨佔一行（帶圓點），下方每條讀音獨立成行：IPA 居左、註釋居右。

### V0.1.3 (2025.07.23)
修復升級後資料庫未更新導致查詢崩潰的問題。

新增資料庫版本檢查機制：檢測到舊版資料庫時自動刪除並從 assets 重新拷貝，確保 schema 升級（排序列）生效。

### V0.1.2 (2025.07.23)
搜尋結果按漢字音典「音典排序」排列各方言點讀音。

資料庫新增「排序」列，存儲各方言點的音典排序號（如 I9B-008），卡片內各方言點按此序號順序展示。

### V0.1.1 (2025.07.23)
搜尋結果按字組分組：相同字組在不同方言點的讀音合併到同一張卡片展示。

卡片重排為兩段式結構：頂部顯示字組標題及方言點數，下方逐行列出各方言點讀音。

### V0.1.0 (2025.07.22)
湘典首個版本。

基於漢字音典（MCPDict）字表資料，構建湖南及周邊地區方言讀音資料庫。

功能：
- 搜字：輸入漢字查詢各方言點讀音
- 搜音：輸入 IPA 讀音反查漢字
- 搜釋義：通過釋義關鍵詞反查漢字
- 搜尋結果以卡片展示，含漢字、方言名稱、IPA 讀音及註釋

資料庫：
- 收錄 464 個方言點
- 117,645 組同音字
- 逾 60 萬條讀音記錄
- 覆蓋湘語全部五片及周邊地區

技術：
- Kotlin + Jetpack Compose + Material 3
- SQLite FTS5 全文檢索，離線使用
- 最低支持 Android 7.0（API 24）
